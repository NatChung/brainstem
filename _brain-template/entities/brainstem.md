# brainstem

你正在用的這套知識圖譜引擎。裝好引擎後 `brainstem init` 開一顆腦、用 Claude Code 餵來源,它把每則素材萃取成原子 note 並在餵入當下就建立雙向連結,長成一個照你思考方式生長的第二大腦。

> 這是 onboarding 示範頁,看懂後可刪。
